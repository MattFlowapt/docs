## Quick Example

<CodeGroup>
```javascript JavaScript
// Send a WhatsApp message
const response = await fetch('https://api.flowmod.ai/message/sendText/your-instance', {
  method: 'POST',
  headers: {
    'apikey': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    number: '1234567890',
    text: 'Hello from FlowMod API!'
  })
});
```

```python Python
import requests

# Send a WhatsApp message
response = requests.post(
    'https://api.flowmod.ai/message/sendText/your-instance',
    headers={
        'apikey': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'number': '1234567890',
        'text': 'Hello from FlowMod API!'
    }
)
```

```bash cURL
curl -X POST 'https://api.flowmod.ai/message/sendText/your-instance' \
  -H 'apikey: YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "number": "1234567890",
    "text": "Hello from FlowMod API!"
  }'
```
</CodeGroup>

## Getting Started
