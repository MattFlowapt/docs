## What is FlowIQ?

FlowIQ is an enterprise-grade API platform that provides:
- **Complete conversation history** - Access all WhatsApp messages, including text, media, and metadata
- **Contact management** - Search, filter, and manage your WhatsApp contacts
- **Real-time data access** - Retrieve the latest conversation data via REST API
- **Advanced search** - Find specific messages and contacts quickly
- **Analytics ready** - Structure data for business intelligence and reporting

## Use Cases

### Customer Support Analytics
Build dashboards to analyze support conversations, response times, and customer satisfaction.

```javascript
// Example: Fetch all conversations for analytics
const response = await fetch(
  'https://api.flowiq.live/conversations?tenantId=YOUR_TENANT_ID&whatsappNumber=27123456789&limit=100',
  {
    headers: { 'Authorization': 'Bearer YOUR_TOKEN' }
  }
);

const data = await response.json();
// Analyze message patterns, response times, etc.
```

### CRM Integration
Sync WhatsApp conversations with your CRM system automatically.

```python
# Example: Sync contacts to CRM
import requests

contacts = requests.get(
    'https://api.flowiq.live/conversations?action=contacts&tenantId=YOUR_TENANT_ID',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
).json()

for contact in contacts['contacts']:
    sync_to_crm(contact)
```

### Automated Reporting
Generate daily, weekly, or monthly reports of conversation metrics.

```javascript
// Example: Generate daily report
async function generateDailyReport(tenantId, date) {
  const conversations = await fetchConversationsByDate(tenantId, date);
  
  const metrics = {
    totalMessages: conversations.length,
    byType: countBySenderType(conversations),
    mediaMessages: countMediaMessages(conversations),
    averageResponseTime: calculateResponseTime(conversations)
  };
  
  return generateReport(metrics);
}
```

## API Capabilities

### 1. Conversation Messages

Retrieve complete message history with rich metadata:

```json
{
  "message": "Hello! How can I help you?",
  "sender_type": "bot-whatsapp",
  "created_at": "2025-01-10T14:30:00Z",
  "message_status": "read",
  "media_type": "image",
  "media_url": "https://example.com/image.jpg",
  "reactions": [{"emoji": "üëç", "sender": "27123456789"}],
  "voice_note_transcription": null,
  "assignee": "user-uuid-123"
}
```

**Key Parameters:**
- `tenantId` - Your FlowIQ tenant identifier
- `whatsappNumber` - The WhatsApp number for the conversation
- `limit` - Number of messages per page (default: 10)
- `page` - Page number for pagination
- `search` - Search query to filter messages

### 2. Contact Listing

Get all contacts with pagination and search:

```json
{
  "contact_name": "John Doe",
  "whatsapp_number": "27123456789",
  "last_message": "Thanks for your help!",
  "last_message_time": "2025-01-10T15:45:00Z",
  "unread_count": 2,
  "profile_picture_url": "https://example.com/profile.jpg"
}
```

**Key Parameters:**
- `action=contacts` - Specifies contact listing action
- `tenantId` - Your FlowIQ tenant identifier
- `limit` - Number of contacts per page
- `page` - Page number
- `search` - Search by name or phone number

### 3. Phone Lookup

Find a specific contact by phone number:

```json
{
  "success": true,
  "contact": {
    "contact_name": "Jane Smith",
    "whatsapp_number": "27987654321",
    "profile_picture_url": "https://example.com/profile.jpg"
  }
}
```

**Key Parameters:**
- `action=find-by-phone` - Specifies phone lookup action
- `tenantId` - Your FlowIQ tenant identifier
- `phoneNumber` - WhatsApp number to lookup

## Authentication

All API requests require Bearer token authentication:

```bash
Authorization: Bearer YOUR_BEARER_TOKEN
```

<Warning>
  **Security Best Practices:**
  - Never expose your bearer token in client-side code
  - Use environment variables to store credentials
  - Implement token rotation regularly
  - Monitor API usage for suspicious activity
</Warning>

### Getting Your Credentials

1. **Login** to [app.flowiq.live](https://app.flowiq.live)
2. Navigate to **Settings ‚Üí API Keys**
3. **Generate** or copy your bearer token
4. Find your **Tenant ID** in account settings

## Rate Limits & Performance

FlowIQ API implements rate limiting to ensure fair usage and optimal performance:

- **Default Rate Limit:** 100 requests per minute
- **Burst Allowance:** 150 requests per minute for short bursts
- **Rate Limit Headers:** Included in all responses

```
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 95
X-RateLimit-Reset: 1641492000
```

## Quick Start Example

Here's a complete example to get you started:

<CodeGroup>
```javascript JavaScript
const FlowIQ = {
  baseUrl: 'https://api.flowiq.live',
  token: 'YOUR_BEARER_TOKEN',
  tenantId: 'YOUR_TENANT_ID',

  async getConversations(whatsappNumber, limit = 10, page = 1) {
    const url = `${this.baseUrl}/conversations?tenantId=${this.tenantId}&whatsappNumber=${whatsappNumber}&limit=${limit}&page=${page}`;
    
    const response = await fetch(url, {
      headers: {
        'Authorization': `Bearer ${this.token}`
      }
    });

    return await response.json();
  },

  async getContacts(search = '', limit = 20, page = 1) {
    const url = `${this.baseUrl}/conversations?action=contacts&tenantId=${this.tenantId}&search=${encodeURIComponent(search)}&limit=${limit}&page=${page}`;
    
    const response = await fetch(url, {
      headers: {
        'Authorization': `Bearer ${this.token}`
      }
    });

    return await response.json();
  }
};

// Usage
const conversations = await FlowIQ.getConversations('27123456789');
console.log(conversations.messages);

const contacts = await FlowIQ.getContacts('John');
console.log(contacts.contacts);
```

```python Python
import requests

class FlowIQ:
    def __init__(self, bearer_token, tenant_id):
        self.base_url = 'https://api.flowiq.live'
        self.token = bearer_token
        self.tenant_id = tenant_id
        self.headers = {'Authorization': f'Bearer {bearer_token}'}
    
    def get_conversations(self, whatsapp_number, limit=10, page=1):
        url = f'{self.base_url}/conversations'
        params = {
            'tenantId': self.tenant_id,
            'whatsappNumber': whatsapp_number,
            'limit': limit,
            'page': page
        }
        response = requests.get(url, headers=self.headers, params=params)
        return response.json()
    
    def get_contacts(self, search='', limit=20, page=1):
        url = f'{self.base_url}/conversations'
        params = {
            'action': 'contacts',
            'tenantId': self.tenant_id,
            'search': search,
            'limit': limit,
            'page': page
        }
        response = requests.get(url, headers=self.headers, params=params)
        return response.json()

# Usage
api = FlowIQ('YOUR_BEARER_TOKEN', 'YOUR_TENANT_ID')

conversations = api.get_conversations('27123456789')
print(conversations['messages'])

contacts = api.get_contacts('John')
print(contacts['contacts'])
```
</CodeGroup>
