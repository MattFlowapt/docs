# Messaging

Send various types of WhatsApp messages including text, images, documents, audio, and more.

## Send Text Message

<Endpoint method="POST" endpoint="/message/sendText/{instanceName}">
  Send a text message to a WhatsApp number.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<ParamField body="number" type="string" required>
  Recipient's phone number (international format without +)
</ParamField>

<ParamField body="text" type="string" required>
  The text message to send
</ParamField>

<ParamField body="delay" type="number">
  Optional delay in milliseconds before sending
</ParamField>

### Response

<ResponseField name="key" type="object">
  Message identifier object
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.flowmod.ai/message/sendText/my-instance" \
  -H "apikey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "number": "1234567890",
    "text": "Hello from FlowMod API!"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/message/sendText/my-instance', {
  method: 'POST',
  headers: {
    'apikey': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    number: '1234567890',
    text: 'Hello from FlowMod API!'
  })
});

const result = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.flowmod.ai/message/sendText/my-instance',
    headers={
        'apikey': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'number': '1234567890',
        'text': 'Hello from FlowMod API!'
    }
)
```
</CodeGroup>

## Send Media Message

<Endpoint method="POST" endpoint="/message/sendMedia/{instanceName}">
  Send media files (images, videos, documents, audio) with optional caption.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<ParamField body="number" type="string" required>
  Recipient's phone number (international format without +)
</ParamField>

<ParamField body="mediatype" type="string" required>
  Type of media: "image", "video", "document", "audio"
</ParamField>

<ParamField body="media" type="string" required>
  URL or base64 encoded media file
</ParamField>

<ParamField body="caption" type="string">
  Optional caption for the media
</ParamField>

<ParamField body="fileName" type="string">
  Optional filename for documents
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.flowmod.ai/message/sendMedia/my-instance" \
  -H "apikey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "number": "1234567890",
    "mediatype": "image",
    "media": "https://example.com/image.jpg",
    "caption": "Check this out!"
  }'
```

```javascript JavaScript
await fetch('https://api.flowmod.ai/message/sendMedia/my-instance', {
```

