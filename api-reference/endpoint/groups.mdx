# Group Management

Create WhatsApp groups, manage participants, and control group settings through the API.

## Create Group

<Endpoint method="POST" endpoint="/group/create/{instanceName}">
  Create a new WhatsApp group with specified participants.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<ParamField body="subject" type="string" required>
  Group name/subject
</ParamField>

<ParamField body="participants" type="array" required>
  Array of phone numbers to add as participants
</ParamField>

<ParamField body="description" type="string">
  Optional group description
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.flowmod.ai/group/create/my-instance" \
  -H "apikey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "subject": "My API Group",
    "participants": ["1234567890", "0987654321"],
    "description": "Created via FlowMod API"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/group/create/my-instance', {
  method: 'POST',
  headers: {
    'apikey': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    subject: 'My API Group',
    participants: ['1234567890', '0987654321'],
    description: 'Created via FlowMod API'
  })
});
```

```python Python
response = requests.post(
    'https://api.flowmod.ai/group/create/my-instance',
    headers={
        'apikey': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'subject': 'My API Group',
        'participants': ['1234567890', '0987654321'],
        'description': 'Created via FlowMod API'
    }
)
```
</CodeGroup>

## Add Participants

<Endpoint method="PUT" endpoint="/group/updateParticipant/{instanceName}">
  Add participants to an existing WhatsApp group.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<ParamField body="groupJid" type="string" required>
  Group ID (JID format)
</ParamField>

<ParamField body="action" type="string" required>
  Action to perform: "add", "remove", "promote", "demote"
</ParamField>

<ParamField body="participants" type="array" required>
  Array of participant phone numbers
</ParamField>

<CodeGroup>
```bash cURL
curl -X PUT "https://api.flowmod.ai/group/updateParticipant/my-instance" \
  -H "apikey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "groupJid": "group-id@g.us",
    "action": "add",
    "participants": ["1234567890"]
  }'
```

```javascript JavaScript
await fetch('https://api.flowmod.ai/group/updateParticipant/my-instance', {
  method: 'PUT',
  headers: {
    'apikey': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    groupJid: 'group-id@g.us',
    action: 'add',
    participants: ['1234567890']
  })
});
```

```python Python
requests.put(
    'https://api.flowmod.ai/group/updateParticipant/my-instance',
    headers={
        'apikey': 'YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'groupJid': 'group-id@g.us',
        'action': 'add',
        'participants': ['1234567890']
    }
)
```
</CodeGroup>

## Update Group Info

<Endpoint method="PUT" endpoint="/group/updateGroupInfo/{instanceName}">
  Update group information such as name and description.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<ParamField body="groupJid" type="string" required>
  Group ID (JID format)
</ParamField>

<ParamField body="action" type="string" required>
  Action: "subject" or "description"
</ParamField>

<ParamField body="subject" type="string">
  New group name (when action is "subject")
</ParamField>

<ParamField body="description" type="string">
  New group description (when action is "description")
</ParamField>

<CodeGroup>
```bash cURL
curl -X PUT "https://api.flowmod.ai/group/updateGroupInfo/my-instance" \
  -H "apikey: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "groupJid": "group-id@g.us",
    "action": "subject",
    "subject": "New Group Name"
  }'
```

```javascript JavaScript
await fetch('https://api.flowmod.ai/group/updateGroupInfo/my-instance', {
  method: 'PUT',
  headers: {
    'apikey': 'YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    groupJid: 'group-id@g.us',
    action: 'subject',
    subject: 'New Group Name'
  })
});
```
</CodeGroup>

## Get Group Info

<Endpoint method="GET" endpoint="/group/participants/{instanceName}">
  Get information about a group including participants list.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<ParamField query="groupJid" type="string" required>
  Group ID (JID format)
</ParamField>

### Response

<ResponseField name="participants" type="array">
  Array of participant objects with details
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.flowmod.ai/group/participants/my-instance?groupJid=group-id@g.us" \
  -H "apikey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/group/participants/my-instance?groupJid=group-id@g.us', {
  headers: {
    'apikey': 'YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data.participants);
```
</CodeGroup>
