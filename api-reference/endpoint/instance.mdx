# Instance Management

Monitor your WhatsApp instance connection status and manage QR code generation for linking devices.

## Get Connection State

<Endpoint method="GET" endpoint="/instance/{instanceName}/connectionState">
  Check the current connection status of your WhatsApp instance.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

### Response

<ResponseField name="instance" type="object">
  <Expandable title="Instance Object">
    <ResponseField name="instanceName" type="string">
      The name of the instance
    </ResponseField>
    <ResponseField name="state" type="string">
      Connection state: "open", "connecting", "close"
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.flowmod.ai/instance/my-instance/connectionState" \
  -H "apikey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/instance/my-instance/connectionState', {
  headers: {
    'apikey': 'YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data.instance.state);
```

```python Python
import requests

response = requests.get(
    'https://api.flowmod.ai/instance/my-instance/connectionState',
    headers={'apikey': 'YOUR_API_KEY'}
)

data = response.json()
print(data['instance']['state'])
```
</CodeGroup>

## Generate QR Code

<Endpoint method="GET" endpoint="/instance/{instanceName}/connect">
  Generate a QR code for connecting WhatsApp to your instance.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

### Response

<ResponseField name="qrcode" type="string">
  Base64 encoded QR code image
</ResponseField>

<ResponseField name="count" type="number">
  QR code generation attempt count
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.flowmod.ai/instance/my-instance/connect" \
  -H "apikey: YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/instance/my-instance/connect', {
  headers: {
    'apikey': 'YOUR_API_KEY'
  }
});

const data = await response.json();
// Display QR code: data.qrcode (base64 image)
```
</CodeGroup>

## Restart Instance

<Endpoint method="PUT" endpoint="/instance/{instanceName}/restart">
  Restart a WhatsApp instance connection.
</Endpoint>

### Parameters

<ParamField path="instanceName" type="string" required>
  Your FlowMod instance name
</ParamField>

<CodeGroup>
```bash cURL
curl -X PUT "https://api.flowmod.ai/instance/my-instance/restart" \
  -H "apikey: YOUR_API_KEY"
```

```javascript JavaScript
await fetch('https://api.flowmod.ai/instance/my-instance/restart', {
  method: 'PUT',
  headers: {
    'apikey': 'YOUR_API_KEY'
  }

```
</CodeGroup>
