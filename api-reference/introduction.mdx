## FlowMod WhatsApp API

The FlowMod API provides programmatic access to WhatsApp messaging, group management, and webhook functionality through your connected instance.

<Note>
Your instance is set up and managed at [app.flowiq.live/flowmod](https://app.flowiq.live/flowmod). Once connected, use these API endpoints to integrate WhatsApp into your applications.
</Note>

## Authentication

All API endpoints require authentication using Bearer tokens in the Authorization header:

```bash
Authorization: Bearer YOUR_API_KEY
```

Example API call:
```bash
curl -X GET "https://api.flowmod.ai/instance/YOUR_INSTANCE/connectionState" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

## Core Endpoints

- **Instance Management**: Check connection status and manage your WhatsApp instance
- **Messaging**: Send text messages, media, and handle message operations  
- **Groups**: Create and manage WhatsApp groups programmatically
- **Webhooks**: Configure real-time event notifications for your applications

## Getting Started

1. Get your instance name and API key from [app.flowiq.live/flowmod](https://app.flowiq.live/flowmod)
2. Ensure your WhatsApp is connected (status should be "open")
3. Start making API calls using the endpoints below
