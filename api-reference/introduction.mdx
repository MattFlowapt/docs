# FlowMod API Reference

FlowMod provides a comprehensive REST API for integrating WhatsApp functionality into your applications. Send messages, manage groups, handle webhooks, and more.

## Base URL

All API requests should be made to:

```
https://api.flowmod.ai
```

## Authentication

All API endpoints require authentication using an API key in the header:

```bash
apikey: YOUR_INSTANCE_API_KEY
```

### Example Request

```bash
curl -X GET "https://api.flowmod.ai/instance/YOUR_INSTANCE/connectionState" \
  -H "apikey: YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

## Response Format

All API responses follow a consistent format:

```json
{
  "success": true,
  "data": {
    // Response data here
  },
  "error": null
}
```

### Error Responses

Error responses include details about what went wrong:

```json
{
  "success": false,
  "data": null,
  "error": {
    "code": "INVALID_REQUEST",
    "message": "Instance not found"
  }
}
```

## Rate Limits

FlowMod API respects WhatsApp's natural usage patterns. There are no artificial rate limits imposed by our API, but we recommend following WhatsApp's guidelines for message frequency.
