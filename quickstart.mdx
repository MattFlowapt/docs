# Get Started with FlowMod API

Transform your personal WhatsApp into a powerful API in just **5 minutes**. No business verification, no complex setup - just scan, authenticate, and start sending messages programmatically.

## Step 1: Get Your API Credentials

### üîë Sign Up & Get API Key
First, get your FlowMod API credentials:

1. Sign up at [FlowMod API Portal](https://api.flowmod.ai)
2. Create a new project
3. Copy your API key from the dashboard

<Warning>
Keep your API key secure and never expose it in client-side code. Use environment variables or secure credential storage.
</Warning>

## Step 2: Connect Your WhatsApp Instance

### üîó Create Connection
Create a new WhatsApp instance using the API:

<CodeGroup>
```bash cURL
curl -X POST 'https://api.flowmod.ai/instance/create' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "instanceName": "my-whatsapp-api",
    "integration": "WHATSAPP-BAILEYS"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/instance/create', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    instanceName: 'my-whatsapp-api',
    integration: 'WHATSAPP-BAILEYS'
  })
});

const data = await response.json();
console.log('Instance created:', data);
```

```python Python
import requests

response = requests.post(
    'https://api.flowmod.ai/instance/create',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'instanceName': 'my-whatsapp-api',
        'integration': 'WHATSAPP-BAILEYS'
    }
)

data = response.json()
print('Instance created:', data)
```
</CodeGroup>

### üì± Get QR Code
Generate a QR code to link your personal WhatsApp:

<CodeGroup>
```bash cURL
curl -X GET 'https://api.flowmod.ai/instance/my-whatsapp-api/connect' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/instance/my-whatsapp-api/connect', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
// QR code will be in data.qrcode (base64 image)
console.log('QR Code:', data.qrcode);
```

```python Python
response = requests.get(
    'https://api.flowmod.ai/instance/my-whatsapp-api/connect',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
# QR code will be in data['qrcode'] (base64 image)
print('QR Code received')
```
</CodeGroup>

### ‚úÖ Scan & Link Your WhatsApp
1. Open WhatsApp on your phone
2. Go to **Settings** ‚Üí **Linked Devices** ‚Üí **Link a Device**
3. Scan the QR code from the API response
4. Your personal WhatsApp is now connected to the API!

<Note>
QR codes expire after 40 seconds. If it expires, simply request a new one using the same endpoint.
</Note>

## Step 3: Verify Connection Status

Check that your WhatsApp instance is connected and ready for API calls:

<CodeGroup>
```bash cURL
curl -X GET 'https://api.flowmod.ai/instance/my-whatsapp-api/connectionState' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

```javascript JavaScript
const response = await fetch('https://api.flowmod.ai/instance/my-whatsapp-api/connectionState', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const status = await response.json();
console.log('Connection status:', status.state); // Should be "open"
```

```python Python
response = requests.get(
    'https://api.flowmod.ai/instance/my-whatsapp-api/connectionState',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

status = response.json()
print('Connection status:', status['state'])  # Should be "open"
```
</CodeGroup>

<Tip>
Wait for the status to show `"open"` before proceeding. This usually takes 10-30 seconds after scanning the QR code.
</Tip>

## Step 4: Send Your First API Message

Now for the exciting part - send your first programmatic WhatsApp message!

### üìù Send Text Message

<CodeGroup>
```bash cURL

  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "number": "1234567890",
    "text": "Hello from FlowMod API! üöÄ"
  }'
```

```javascript JavaScript

  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    number: '1234567890',
    text: 'Hello from FlowMod API! üöÄ'
  })
});
```

## üÜò Troubleshooting

<AccordionGroup>
  <Accordion title="Connection Issues">
    - Ensure your phone has stable internet connectivity
    - Try generating a new QR code if the previous one expired
    - Verify WhatsApp is updated to the latest version
    - Check that you're scanning with the correct WhatsApp account
  </Accordion>
  
  <Accordion title="Message Not Delivered">
    - Verify phone number format (international format without + symbol)
    - Confirm your instance connection state shows "open"
    - Ensure the recipient has WhatsApp installed and is reachable
    - Check that you're not exceeding WhatsApp's natural usage limits
  </Accordion>
  
  <Accordion title="API Authentication Errors">
    - Verify your API key is correct and hasn't expired
    - Check that you're including the "Bearer " prefix in the Authorization header
    - Ensure your API key has the necessary permissions
    - Confirm the API endpoint URLs are correct
  </Accordion>
</AccordionGroup>
